Section I: Bezier Curves and Surfaces

Part 1: Bezier curves with 1D de Casteljau subdivision

Bezier curves take in a set of an unlimited number of ordered points (n) and returns a smooth curve using a recursive subdivision algorithm. To create the curve, linear interpolation is conducted across each set of consecutive points, creating a new set of n-1 interpolated points. This means for some value t in the range [0,1], each interpolated point is a distance proportional to t between two neighboring points. This process is repeated recursively until just one point remains. That point lies on the Bezier curve. By varying t from 0 to 1, the full path of the curve is traced.

Part 2: Bezier surfaces with separable 1D de Casteljau

To create a Bezier surface from an nxn grid of control points, the same idea used for Bezier curves is applied in two directions. First, for each of the n rows of 3D control points, a Bezier curve is evaluated using the parameter u. For any specific value of u, this produces n intermediate points, one from each row. These points are then treated as control points for another Bezier curve in the perpendicular direction, evaluated using a second parameter v. By varying both u and v between 0 and 1, points across the entire Bezier surface are generated.
